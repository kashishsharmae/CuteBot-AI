# 💖 CuteBot AI (Offline Chatbot)

> A beautiful, fully offline AI chatbot built using **Streamlit + Ollama**, featuring a modern UI and local LLM integration.

---

## 🚀 Project Overview

FlexiMode AI Chatbot is a **100% offline conversational AI system** that runs locally on your machine using Ollama models. It provides a smooth chat experience with a **cute magenta-themed UI**, chat history management, and fast responses without any API dependency.

This project is designed for:
- Offline AI usage
- Privacy-focused chat applications
- Learning LLM integration
- Streamlit UI development practice

---

## ✨ Key Features

- 💬 Fully offline AI chatbot (no internet required after setup)
- ⚡ Powered by Ollama local LLMs (Gemma / LLaMA / Mistral)
- 🎨 Modern magenta gradient UI
- 🙈 Hide/Show chat history toggle
- 🧠 Real-time local AI responses
- 💻 Lightweight Streamlit web app
- 🔒 No API keys or external services required

---

## 🧠 How It Works

1. User enters a message in the Streamlit UI  
2. The message is sent to `chatbot.py`  
3. Ollama local model processes the prompt  
4. Response is generated on your machine  
5. Streamlit displays the chatbot response instantly  

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend Logic:** Python
- **AI Engine:** Ollama (Local LLMs)
- **Model Examples:** Gemma 3, LLaMA 3, Mistral
- **HTTP Requests:** requests library
