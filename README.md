# FlexiMode AI Chatbot

A Dual-Mode AI Chatbot built with Streamlit.

## Features

- Online AI Mode using Google Gemini
- Offline AI Mode using Ollama (Gemma 3 1B)
- Chat History Storage using SQLite
- Modern Streamlit UI

## Technologies

- Python
- Streamlit
- Google Gemini API
- Ollama
- SQLite

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
